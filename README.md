# Predictive Maintenance for Industrial Equipment

## Overview
Machine learning project for predicting Remaining Useful Life (RUL) of turbofan engines using the NASA C-MAPSS dataset.

## Dataset
- **Source**: NASA Prognostics Center - C-MAPSS (Commercial Modular Aero-Propulsion System Simulation)
- **Description**: Run-to-failure simulated data of turbofan jet engines under different operating conditions and fault modes

## Models
- Linear Regression (baseline)
- Random Forest
- XGBoost
- LSTM (Deep Learning)

## Project Structure
```
├── data/               # Raw and processed data

├── src/                # Source code
├── models/             # Saved models
├── results/            # Plots and evaluation results
```
